LeetCode Setup

Manual is the default. That means after a real accepted submit, you choose when to push to GitHub.

Use the token that has access to this exact repository.
Manual shows the page button after acceptance. Automatic pushes immediately after acceptance.

Before You Enable Sync

CodeSync reads accepted-solution code, problem metadata, and your visible LeetCode username from LeetCode problem pages so it can prepare a GitHub sync.

CodeSync sends your solution code and selected repository details to the GitHub API only to create or update files in the repository you choose.

Your settings, repository details, token, and sync preference are stored in extension storage. Use a token that has access only to the repository you want to sync.

You're set

CodeSync is ready for LeetCode. You can come back here anytime to change your repo, token, username, or sync mode. Sync only works for the matching LeetCode username you configured.
